Позиционирование в CSS

position — свойство, определяющее тип позиционирования элемента. Основных значений 4: static (по умолчанию), relative, absolute, fixed.

🔸 position: relative;
Этот блок сдвинут влево на 15px относительно своего обычного положения.

При absolute и fixed элемент выпадает из потока. Fixed привязывается к окну браузера.

Z-index управляет порядком слоёв.

Таблица значений

Значение Характеристика
static обычный поток, смещения не работают
relative смещение относительно себя
absolute относительно ближайшего позиционированного предка
fixed относительно окна браузера
.menu {
    position: fixed;
    top: 0;
    width: 100%;
}

Свойство z-index задаёт порядок перекрытия элементов.
